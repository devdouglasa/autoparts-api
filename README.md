# Autoparts API

## REST API for inventory control in an auto parts store made with Nestjs.

### ➦ Technologies Used:

- NestJS
- Typescript
- JWT (JSON Web Token)
- Prisma ORM
- PostgreSQL
- Docker
- Git
- Github

### ➦ Running the Application:

In your terminal (bash/powershell/cmd), type the following commands:

You will need to have Git installed on your machine.

```
git clone https://github.com/devdouglasa/autoparts-api.git
```

Enter the project folder:
```
cd autoparts-api
```
You will need to have Docker and Docker Compose installed on your machine.
```
docker compose up --build
```

That's it! Your application will start building and will be available at: http://localhost:3000/ <br>
